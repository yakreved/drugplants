== Требования

Тема: Проектирование базы данных и разработка (web) приложения обслуживания  БД (предметная область).

Технические требования
 -     инфологическое проектирование должно быть выполнено с помощью  CASE- средства ERWin, 
-      на этапе логического проектирования  обязательна проверка на соответствие всех таблиц нормальным формам отношений,
на этапе физического  проектирования обязательна оценка эксплуатационных характеристик,
разработанное приложение должно выполнять функции ввода, добавления, удаления, корректировки данных,  поиска данных по ключу, сервисные функции, формирования отчетов,
-     использование объектно-ориентированного подхода,
использование операторов языка SQL,
запросы к БД должны быть оптимизированы,
во время ввода и изменений базы данных должны проводиться проверки на целостность данных,
организация интерфейса пользователя должна отвечать требованиям стандарта среды Windows,
количество записей тестовых примеров базы данных должно быть достаточно для полного тестирования приложения и оценки эксплуатационных характеристик.
